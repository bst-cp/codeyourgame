# Code Your Game
"Code Your Game" is a mobile game that built with the aim to teach programming to the kids. Kids would program their games and they will learn programming while learning some programming basics. Variables, conditional statements, loops and nested code structures are main teaching concepts of this game.

If you want to check out the game without installing it to your android smartphone, you can directly access it's web version via https://bst-cp.github.io/codeyourgame .
